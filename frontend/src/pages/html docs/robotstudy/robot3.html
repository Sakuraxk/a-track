分类 Classification

将数据划分到已知的类别中。需要带标签的训练数据。

典型算法：决策树、SVM、KNN、逻辑回归

应用：垃圾邮件检测、图像识别、疾病诊断

聚类 Clustering

将数据自动分组，发现数据内在结构。无需标签。

典型算法：K-Means、DBSCAN、层次聚类

应用：用户分群、图像分割、异常检测

SVM 核心思想

寻找一个超平面，使得两类样本到平面的最小距离（间隔）最大化。

支持向量

距离分类边界最近的样本点，决定了最终的分界面位置。

如何选择算法？

  • 需要可解释性 -> 决策树
  • 数据量小、维度高 -> SVM
  • 数据量大、实时预测 -> KNN不适合
  • 无标签数据 -> K-Means
  • 非线性问题 -> 核SVM或集成树

经验法则：先用简单模型（如逻辑回归、决策树）建立基线，再尝试复杂模型。
