从生物神经元到人工神经元的跨越

生物神经元

生物神经元接收来自其他神经元的信号（树突），在细胞体内整合，当信号超过阈值时，通过轴突向下一个神经元传递电脉冲。

人工神经元（感知机）

人工神经元接收多个输入(x)，每个输入有权重(w)，加权求和后经过激活函数，产生输出。这就是感知机的基本结构。

y = f(Σ w_i·x_i + b)

加权求和 + 偏置 + 激活函数

学习 = 调整权重

让预测更接近真实

感知机 Perceptron

1958年，Frank Rosenblatt 提出了第一个可学习的神经网络

交互式感知机

调节权重和偏置，观察决策边界的变化

权重 w10.80
权重 w2-0.50
偏置 b0.20
y = step( 0.80·x1 + (-0.50)·x2 + 0.20 )

决策边界可视化

决策边界方程: 0.80 x1 - 0.50 x2 + 0.20 = 0

感知机的局限：只能解决线性可分问题。1969年，Minsky和Papert证明了感知机无法解决XOR（异或）问题，导致神经网络研究进入寒冬。

激活函数 Activation Function

没有激活函数，多层网络也只是线性变换

Sigmoid

1/(1+e^-x)

输出范围 (0,1)

优点：平滑，可解释为概率
缺点：梯度消失，输出非零中心

Tanh

(e^x-e^-x)/(e^x+e^-x)

输出范围 (-1,1)

优点：零中心，比Sigmoid好
缺点：仍有梯度消失问题

ReLU

max(0, x)

输出范围 [0, +∞)

优点：计算简单，缓解梯度消失
缺点：神经元可能"死亡"

多层前馈神经网络

堆叠多层神经元，解锁非线性表达能力

交互式网络构建器

隐藏层层数2

网络结构解释

  • 输入层：接收原始数据
  • 隐藏层：提取特征，层数越多表达能力越强
  • 输出层：产生最终预测

前向传播计算

输入: [x1, x2, x3]

隐藏层1: a1 = f(W1·x + b1)

隐藏层2: a2 = f(W2·a1 + b2)

输出: y = f(W3·a2 + b3)

万能近似定理：一个包含足够多神经元的单隐藏层网络，可以逼近任意连续函数。

反向传播 Backpropagation

1986年，Rumelhart等人重新发明了反向传播，神经网络迎来复兴

交互式梯度下降演示

学习率0.10

当前位置: (2.5, 6.25)

梯度: 5.0

链式法则

∂L/∂w = ∂L/∂y · ∂y/∂a · ∂a/∂w

反向传播步骤

  1. 前向传播计算输出
  2. 计算损失函数
  3. 反向计算每层梯度
  4. 用梯度更新权重
