人工智能 / 机器学习 / 深度学习
理解三者之间的包含与演进关系

人工智能 (Artificial Intelligence)：一个广泛的领域，旨在创建能够模拟人类智能行为的系统。它涵盖了所有让机器具备感知、推理、学习和决策能力的技术。

机器学习 (Machine Learning)：人工智能的核心子集。其核心思想是让机器从数据中自动学习规律，而不需要被显式地编程每一个规则。

深度学习 (Deep Learning)：机器学习的一个分支，使用多层神经网络来学习数据的层次化表示。它在图像识别、自然语言处理等领域取得了突破性进展。

Alex Krizhevsky设计的AlexNet在ImageNet图像分类竞赛中大幅领先传统方法，错误率从26%降至16%，深度学习时代正式开启。

分类任务：预测离散的类别标签。模型学习将输入数据划分到预定义的类别中。

无监督学习详解
在没有标签的数据中发现隐藏的结构与模式

聚类 (Clustering)：将数据自动分组为若干簇，使得同一簇内的数据点彼此相似，不同簇的数据点差异较大。常用算法包括K-Means、DBSCAN、层次聚类。

降维 (Dimensionality Reduction)：在保留数据核心信息的前提下，将高维数据映射到低维空间。常用方法有PCA、t-SNE、UMAP。

异常检测 (Anomaly Detection)：识别数据中与大多数模式显著不同的样本。在欺诈检测、设备故障预警等场景中广泛应用。

强化学习详解
智能体通过与环境交互、试错来学习最优策略 -- 点击方向键控制代理

核心要素：

智能体 (Agent)：做出决策的主体，通过观察环境状态来选择行动。

环境 (Environment)：智能体所处的外部世界，会对智能体的行为做出响应。

奖励 (Reward)：环境反馈的信号，用于评估行为的好坏。智能体的目标是最大化长期累积奖励。

核心公式 (贝尔曼方程)：

Q(s,a) = R(s,a) + gamma * max Q(s',a')

数据收集 (Data Collection)
机器学习的起点是获取高质量的训练数据。数据来源包括：数据库查询、API接口、网络爬虫、传感器采集、公开数据集（如Kaggle、UCI）等。数据的质量和数量直接决定了模型性能的上限。一个好的经验法则是：数据量至少应该是模型参数量的10倍以上。在收集阶段，需要关注数据的代表性、完整性和时效性。

损失函数与优化目标
损失函数衡量模型预测与真实值之间的差距 -- 拖动滑块观察损失变化

损失函数 (Loss Function)是机器学习的核心组件。它量化了模型预测值与真实标签之间的差异，模型训练的目标就是找到使损失函数最小化的参数。

均方误差 (MSE): L = (1/n) * sum((yi - f(xi))^2)
交叉熵 (Cross-Entropy): L = -sum(yi * log(f(xi)))

选择原则：回归任务通常使用MSE；分类任务通常使用交叉熵。损失函数必须是可微的，以便通过梯度下降进行优化。
