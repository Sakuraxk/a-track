Motivation

程序需要"做决定"

没有控制流的程序只能从上到下逐行执行。控制流让程序能根据条件走不同的路，或者重复做某件事。

条件判断
如果下雨，就带伞；否则穿短袖。
这就是 if-else。
weather = "rain" if weather == "rain": print("带伞") else: print("穿短袖")
循环结构
每天早上重复：起床、刷牙、吃饭。
直到放假才停止。这就是 循环。
days = ["周一","周二"] for day in days: print(f"{day}: 起床")

Part A

if 语句

if 是最基本的条件判断。条件为 True 时执行缩进代码块。

Python 用缩进（4个空格）标记代码块，不用花括号。
temp = 35 if temp > 30: print("太热了")
