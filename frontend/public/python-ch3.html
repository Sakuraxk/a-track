函数与模块

封装复用 · 模块化编程

函数是组织代码的基本单元。你可以将重复使用的代码封装成函数，通过参数传递数据，并获得返回值。模块则让你能够使用他人写好的强大功能。

本章你将学会：如何定义和调用函数、理解各种参数类型、掌握返回值与作用域，以及如何使用 math、random、datetime 等常用模块。

定义与调用函数

使用 def 关键字创建函数

基本语法

def 函数名(参数): """文档字符串""" 函数体 return 返回值

函数定义后不会立即执行，需要调用才会运行。

交互：定义并调用函数

def greet(name):
    return f"Hello, {name}!"
调用结果: Hello, Python!

函数命名规范：小写字母，单词间用下划线分隔。函数体必须缩进。

参数传递方式

参数类型

  • 位置参数 — 按顺序传递
  • 关键字参数 — 指定参数名
  • 默认参数 — 提供默认值
  • *args — 可变位置参数
  • **kwargs — 可变关键字参数

交互演示：计算器函数

10 + 5 = 15
默认乘法: 10 * 2 = 20

*args 与 **kwargs 示例

sum(1,2,3,4) = 10
{'name':'Alice','age':25}

返回值与变量作用域

return 语句

函数可以返回一个或多个值。若无 return 或 return 不带值，则返回 None。

def add(a, b): return a + b

作用域 LEGB

Local -> Enclosing -> Global -> Built-in

函数内部变量为局部，外部无法访问。使用 global 可声明全局变量。

导入方式

math 模块计算器
