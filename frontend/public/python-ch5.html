类与对象：封装数据与行为

定义类

class Dog: def __init__(self, name): self.name = name def bark(self): return f"{self.name} says woof!"

交互：创建你的宠物

当前对象: Dog('Buddy')
叫声: Buddy says woof!

self 代表实例本身。__init__ 是构造方法，在创建对象时自动调用。

异常处理：try/except

语法结构

try: num = int(input) except ValueError: print("无效数字") else: print("成功") finally: print("结束")

综合示例：银行账户类

Account 类

class Account: def __init__(self, owner, balance=0): self.owner = owner self.balance = balance def deposit(self, amount): self.balance += amount def withdraw(self, amount): if amount > self.balance: raise ValueError("余额不足") self.balance -= amount
