# Changelog

本文件记录项目的重要版本变更。

## [1.0.0] - 2026-05-11

### 🎉 初始开源版本

首次公开发布，包含完整的 AI 多学科自适应学习平台核心功能。

### Added

- **智能水平评估**：快速测验生成能力画像，精准定位知识薄弱点
- **多学科沉浸式学习**：支持 Python 编程、机器学习、高等数学、概率论、线性代数、统计学、AI 通识 7 大学科
- **动态知识图谱**：基于学科特性构建可视化知识树与前置依赖
- **苏格拉底 AI 导师**：引导式教学，结合用户记忆流提供个性化提示
- **用户记忆系统**：跨学习周期追踪行为、偏好与学习模式
- **交互式练习环境**：前端轻量执行 + 后端独立沙箱执行 + 多样化题型练习
- **智能题库**：AI 生成题目，按难度/知识点自动编排
- **概念学习工作台**：流式生成学习内容，内嵌 AI 提问与 SVG 配图
- **AI 学习路线规划**：AI 生成多阶段个性化学习计划与每日任务
- **成就树系统**：章节级进度可视化与成就解锁
- **学习社区**：帖子分享、点赞、评论与互动通知
- **能力雷达图**：多维度可视化展示学习进度与成就
- **Prompt 管理系统**：Prompt 编辑、版本管理与调试工具
- **数学实验室**：交互式数学公式计算、JSXGraph 函数图形绘制
- **管理后台**：独立 Vue 3 管理面板，用户/学科/社区管理与数据分析
- **全栈 Docker 容器化部署**：一键启动开发和生产环境
